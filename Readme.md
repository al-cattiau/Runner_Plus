# Runner Plus 文档

开发环境相关：

1. 安装 Cocoa Pods，在项目根目录执行 pod install，安装开发相关的第三方库依赖。
2. 安装最新版 macOS 以及 xCode，双击 Runner_Plus.xcworkspace 打开项目。
3. 选择任意一个 iOS 11 以上的模拟器，点击编译运行，体验 App。
